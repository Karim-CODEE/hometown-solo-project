# Egypt Travel Guide 🇪🇬

A responsive landing page showcasing the beauty and culture of Egypt. This project was built to practice modern web design principles, including Flexbox and responsive layouts.

## ✨ Features
* **Full-screen Hero Section:** Utilizing CSS Viewport units for a cinematic feel.
* **Responsive Design:** Optimized for different screen sizes without distortion.
* **Personalized Guide Section:** A custom UI card featuring local insights.

## 🛠️ Built With
* HTML5
* CSS3 (Custom properties and Flexbox)
